ZFS
=========

A role to setup ZFS monitoring on the proxmox host.

Requirements
------------

None.

Role Variables
--------------

None.

Dependencies
------------

This role does not have any dependencies on other Galaxy roles.

Example Playbook
----------------

Here is an example of how to use this role with variables passed in as parameters:

```yaml
- hosts: servers
  roles:
    - { role: zfs }
```
